# MITA-State-Self-Assessment
This repository will be used to research and recommend a reimagined Medicaid Information Technology Architecture (MITA) State Self-Assessment tool that is less burdensome and more useful to states and the Centers for Medicare and Medicaid Services (CMS).

## North Star Statement
We plan to create an SS-A tool which collects state Medicaid maturity inputs and provides an assessment of maturity based on the MITA maturity model. The SS-A tool will guide the user through the maturity assessment process via a series of questions, be easy and intuitive to use, and illustrate areas where maturity could be improved.
